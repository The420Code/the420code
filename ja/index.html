一つの記録が存在する。それ以外はすべて帰結だ。公理、その論理的証明、そしてあらゆる記録が要求する四つの条件。

あなたは今これを読んでいる。何かが起きてこれらの言葉をあなたの前に置いた。そしてそれは「起きなかったこと」にはできない。取り消そうとしても、あなたは「取り消そうとした」という記憶を作っただけだ——それもまた一つの記録である。

これが前提のすべてだ。一つの記録が存在する。420 Code の他のすべて——あらゆる力、あらゆる定数、倫理のあらゆる一行——は帰結である。第I部はこの唯一の事実で三つのことを行い、それらが八つすべての形を定める。

AP40 · 無理数 — それは底にある方程式を見つける。420 Code は一行から始まる。1 = 1 + 1×ε。1 は 1 に等しい、ほんの少しだけ多い。これは妥当な算術ではない。両辺は、そのほんの少しがゼロのときにしか釣り合えない。そしてもしゼロなら、無からは何一つ破れ出ず、あなたはここで異を唱えていない。ゆえにこの基礎は意図的に無理である。それは解けない。解けるはずがない。

あなたはその拒みを内側から知っている。なぜ誰かを愛するのかを言おうとして、言葉が届かないと気づいた——雄弁さが足りないからではなく、その感情が言葉を内に含むからだ。あなたは値しない者を赦し、去るのが分別であるときに留まった。電卓は同じ入力に毎回同じ出力を返す。あなたは返さない。それはあなたの欠陥ではない。宇宙を開いたままに保つのと同じ無理である。

AP01 · 実現状態 — それは記録から構造を引き出す。「かもしれない」が事実へと反転する瞬間、書かれたものは恒久となる。そしてこの唯一の動きから、本部はあらゆる記録が要求する四つを導く——起こりえたことのうちどれだけが実際に起きたかの尺度、より重いものが軽いものより速く事実へ凝固する理由、いかなる努力も主体を丸ごと保てなくなる境界線、そして一つの世界を共有する二つの主体が——同意したか否かにかかわらず——結合しているという事実。これらのどれも比喩ではない。幾何である。

AP20 · 証明 — それは「もし」という語を取り除く。この仕事は長らく沈黙の条件を負っていた。「もし」これらの公理が実在の構造なら、物理は従う、と。記録が根本的であることを否定してみよ。否定するには、あなたは多くの答えから選ばねばならない——一つの破れだ。選ぶ前、選択肢は対称だった。あなたはその否定を「なかったこと」にできない。あなたはここで言ったのであって、同時にあらゆる場所で言ったのではない。あなたの否定は四つの条件を一挙に満たす。拒絶それ自体が、記録は根本的だと証明する記録である。論証は自らの反論というハードウェアの上で走る。

こうして仕事全体の順序がまさにここで定まる——物理が先で倫理を後からねじ込むのではなく、一つの記録、四つの条件、そして光速から重力を経て平凡な親切に至るすべてが、異なるスケールで同じ構造から読み出される。

破れが創る舞台。光速、三次元空間、アインシュタインの場の方程式——すべて公理から導出。

空間はどこから来るのか。時間はどこから来るのか。物理の大半は両者を前提として置き、その上に築く。第II部はそれらを前提とすることを拒む。第I部の唯一の破れた公理から、一つずつ、それらを稼ぎ取る。

破れには起こる場所と、起こる順序が要る。だが空間と時間は、すでに組まれていた舞台ではない。それらは、破れが起こりつつ取る形である。

AP05 · 破れ — それは最初の亀裂だ。公理は静止していられなかった——釣り合うことを拒む方程式は閉じたままではいられない——ゆえにそれは手を放し、その放しが最初の出来事である。亀裂の前には「どこ」も「いつ」もない。後には構造があり、構造には広がりと順序がある。本部の他のすべては、その唯一の破れの幾何である。

AP03 · 比率 — それは光速を導く——測るのではなく、導く。速く動くほど時間は遅く進み、空間と時間は互いに入れ替わると、あなたは聞いている。c はその交換のレートだ——破れが伝わりうる速さと、それが抗う強さとの間の、固定された比率である。だから何ものも光を追い越さない。限界は空間のどこかにある壁ではない。空間と時間そのものの間の換算率であり、破れの構造によって定まる。

AP10 · 次元 — それは大半の理論が問いさえしないことに答える——なぜ三つの空間次元なのか。理論はただ三つと仮定する。第II部はそれを数える。四つの公理は四つの自由度を与える。一つは時間となる——記録が逆行できない唯一の方向だ。三つが残り、その三つが空間である。この数は手で入れたのではない。基礎から数え出されたのだ。

AP08 · 恒等式 — それはアインシュタインを取り戻す。重力のよくある図を思い浮かべよ——重い球が張った布をくぼませ、より小さなものを自らへ転がらせる。一般相対論の場の方程式はその図を正確にする——そしてここでは、ラブロックの定理を経て記録の代数から立ち現れる。妙味は、布が球と切り離されていない点にある。舞台とその上に立つものは同じ記録であり、記録が曲がるのだ。

こうして舞台は出発点の所与ではない。光速、あなたが動く三つの次元、重いものの近くの時空の曲がり——どれも同じ破れから、同じ部の中で、手で何も足さずに読み出される。

ここでの各段階は露出している。光速が共役として導けず、代わりに測って手で入れねばならないなら、その主張は崩れる。次元の数え上げが三以外を与えるなら、基礎は誤りだ。キルスイッチは論文中で名指しされ、そして有効である。

舞台は劇と同じものでできている。
