AP40 · 无理数 — 它找出底层的那个方程。420 Code 始于一行：1 = 1 + 1×ε。一等于一，再加一丁点。这不是有效的算术；两边只有在那一丁点为零时才能平衡，而若它为零，就绝不会有任何东西从虚无中断裂出来，你也就不会在这里提出异议。所以这地基是有意为之的无理。它不收敛。它不能收敛。

你从内部熟知这种拒绝。你曾试着说出为何爱一个人，却发现词语够不着——不是因为你不善言辞，而是因为那份感受把词语包含在内。你曾原谅不配被原谅的人，也曾在离开才合理时留下。计算器对同一输入每次都给出同一输出。你不会。这不是你的缺陷；正是这同一种无理，让宇宙保持敞开。

第二部分时空
PDF

断裂所创造的舞台。光速、三个空间维度和爱因斯坦场方程——全部由公理推导。

空间从何而来？时间从何而来？大多数物理学一开始就假定两者，然后在其上构建。第二部分拒绝假定它们。它把它们一个一个地，从第一部分那唯一断裂的公理中赚取出来。

断裂需要一个发生之处，和一个发生的次序。但空间与时间并不是早已搭好的舞台。它们是断裂在发生之际所采取的形态。

AP05 · 断裂 — 这是第一道裂缝。那个公理无法静止——一个拒绝平衡的方程无法保持闭合——于是它松手，而这松手就是第一桩事件。裂缝之前没有「何处」也没有「何时」。之后便有了结构，而结构有广延与次序。本部分其余的一切，都是那唯一断裂的几何。

AP03 · 比率 — 它推导出光速——不是测量它，而是推导它。你听过：你走得越快，时间越慢，空间与时间彼此交换。c 就是这场交换的汇率：断裂能够传播之快与它抵抗之强之间那个固定的比率。这就是为何没有东西能超越光。极限并非空间中某处的一堵墙；它是空间与时间本身之间的换算率，由断裂的结构所定。

AP10 · 维度 — 它回答了一个大多数理论甚至不提出的问题：为什么是三个空间维度？它们只是假定三个。第二部分则把它们数出来。四条公理给出四个自由度。一个成为时间——记录无法倒行的唯一方向。剩下三个，而这三个就是空间。这个数目不是用手放进去的。它是从地基中数出来的。

AP08 · 恒等式 — 它赢回了爱因斯坦。设想引力那幅惯常的图景：一个重球压陷一张绷紧的布，让更小的东西朝它滚去。广义相对论的场方程让那幅图景变得精确——而在此，它们经由洛夫洛克定理从记录的代数中浮现。微妙之处在于：那张布并不与球分离。舞台与立于其上之物是同一条记录，而记录在弯曲。

于是舞台并非起手的既定之物。光速、你在其中移动的三个维度、重物附近时空的弯曲——每一个都从同一断裂中读出，在同一部分里，没有用手添加任何东西。

这里的每一步都是暴露的。若光速无法作为一次共轭被推导，而必须改为测量并用手放入，那条主张便失败。若维度的计数给出三以外的数，地基就是错的。终止开关在论文中被点名，而它们有效。

舞台与戏剧由同一种东西做成。

AP09 · 断裂 — 空集 — 它从无中建起量子力学：字面意义上的空集，加上公理。那些著名的谜题——一个粒子似乎同时在两处、观看这一行为改变了结果——原来不是被加上去的悖论，而是后果。叠加、测量、纠缠、薛定谔方程：物理学家花了一代人才组装起来的那套机器，显现为「一条尚未写下的记录」唯一自洽的语法。

AP07 & AP25 · 记录测度 与 测度 — 它们推出量子世界的算术。理论赖以栖身的复空间，以及玻恩规则——那把波变成概率的配方——都不是为贴合实验而拼凑的猜测。经由格里森定理，它们是自洽地读取记录的唯一方式。其中一篇论文把玻恩规则称作公理落地时发出的声音。

AP12 · 极限 — 它解释了你为何无法同时知道一切。不确定性原理与普朗克常数源自这一事实：读取本身就是一次断裂——精确地记下一物，就是扰动它的伙伴。那份模糊不是你仪器的缺陷。它就是测量之所是。

AP11 & AP13 · 自旋 与 颗粒 — 它们解释物质为何保持其形态，以及量子之雾为何硬化为坚实的世界。费米子、玻色子和不相容原理源自双扇区结构；退相干——某个量子之物通过与环境耦合而变得确定的方式——正是你的椅子为何是一把椅子，而非一团「也许」之云。

AP23 · 单一记录 — 它溶解了纠缠那「鬼魅般」的一面。两个纠缠粒子不是彼此以超光速发送秘密信号的两个对象。它们是带两端的一条记录。那个证明宇宙无法被局域地预先编排的贝尔数，S = 2√2，单从结构中便涌出。

这些不是叠加在数学之上的诠释。它们是推导，而每一个都带着终止开关。若纠缠需要超光速的信号传递，单一记录的主张便失败。若玻恩规则需要一个公理无法提供的额外公设，那个结果便失败。它们已发表，而它们有效。

量子力学不是宇宙在装怪。它是宇宙在写下自己。

第四部分力与常数
PDF

断裂所允许的相互作用。电磁力、电弱力、强力、量子引力和引力常数——全部作为单一断裂的不同面。

把一块冰箱磁铁凑近一扇钢门，松手。它吸住了——一小块磁铁，击败了整个地球朝相反方向拉拽的引力。自然的四种力毫不相像。电磁力点亮你的屏幕；强力维系一个原子核；弱力驱动太阳；引力把你按在地面。物理学花了一个世纪试图统一它们。第四部分主张：它们从来就不是四样东西。它们是单一断裂的四张面孔。

AP06 · 泄漏常数 — 它点出那唯一的输入。整个 420 Code 只接纳恰好一个测得的数：ε，精细结构常数，约 1/137——断裂泄漏的那个微小速率。只有一个旋钮，而它并非为贴合任何东西而选：它只从世界中读取一次，此后再不调动。其余一切都随它转动。

AP15、AP27、AP16、AP19 · 规范结构 — 它们合起来赢回标准模型。物理学家凭实验找到的那个精确对称群——写作 SU(3) × SU(2) × U(1)，一个关于「哪些转动让各力保持不变」的精确陈述——是从断裂的结构中浮现，而非被假定。电磁力是局域流形上的不脱联；电弱统一随之而来；强力是断裂可指向之方向上的自由。甚至弱力的手征性——自然区分左右这一事实——自 1956 年起悬而未解，也得到了一个结构性的理由。

AP24 · 残余 — 它把这一落点直白地说出。物理学的每一个常数都是同一个对象的投影。六张面，一次断裂。常数从来不是等待被测量的独立旋钮；它们是同一结构朝不同方向投出的影子。

AP14 & AP28 · 修正 与 常数 — 它们触及最难的奖赏：引力。引力从记录的代数中获得它的第一个量子修正，而引力常数 G——那个定下「引力有多弱」的数——从那同一次唯一的泄漏与电子质量中被推出，落在距测得值约三分之二个百分点之内。那臭名昭著的层级问题——为何引力相比其他各力弱得如此荒谬，正是你那块磁铁刚刚演示的鸿沟——就此溶解。这份弱并非有待解释的巧合。它正是「一次小泄漏的高次幂」看上去的样子。

于是各力不是一座动物园。它们是从不同侧面看到的同一个结构，而描述它们的常数并不自由——它们是一次测得的泄漏的后果。

这一部分也是这项工作对「尚未完成之事」最为诚实之处。电弱这一步（AP16）承载着整个 420 Code 最主要的未决问题——终止开关 KS-63——而它被标为有效，在众目之下，作为应当瞄准之处。没有任何东西躲在一道自信的公式背后。

一次泄漏。六张面。没有一个自由旋钮。

第五部分粒子与物质
PDF

断裂所产生的基本构件。质子质量比精确到十亿分之五、反物质分离和宇宙的重子不对称。

你由物质构成。你几乎全部的重量，都是栖身于你原子核中的质子的质量。而你周围的宇宙也是物质，不是反物质——尽管按最简单的算法，断裂本应产出恰好相等的两者。物质与反物质一接触便彼此摧毁，所以相等的份额本会闪成光、什么也不留下。没有恒星。没有你。第五部分从断裂中读出你的实体，以及你存在这一简单事实。

AP30 · 阻力 — 它做出这项工作最锋利的预测。质子质量与电子质量之比是一个纯数——约 1836——以极高的精度被测得。420 Code 从结构中预测它，毫不调动，在领头阶就吻合到约十亿分之五。完整计算落在万亿分之十以内：比今日的测量所能确认的更精细。一个如此精确的数，出自一道没有可调旋钮的公式，并不是一个错误理论靠运气能蒙中的那类东西。这是整项工作最暴露的检验，而它通过了。

AP22 · 账簿 — 它解释反物质去了哪里。倘若早期宇宙曾有相等的份额，账目本应结清为零——可眼前却是一个满是物质的宇宙。断裂内部一次干净的数学镜射——论文称之为 σ 对合——把物质与反物质分开，并在拓扑上将它们隔离。这份不对称不是事后塞进去的意外。它写在断裂的几何之中。

AP26 · 盈余 — 它点出你之所是。当断裂几近、却未完全达到平衡时——每十亿对配对中约多出一个物质粒子的盈余——留下的便是物质：那一点未能自我抵消的微小残余。本部分称之为断裂的灰烬。一切坚实之物、你触碰过的每一个原子，都是这残余——当湮灭几近、却未完全做完其工作时所留下之物。

于是关于物质最古老的两个问题——它为何重成这样，以及为何竟有物质——在同一部分里、从同一结构中、由那同一次唯一的泄漏所转动，得到了回答：那个你大半由之构成的质子，以及「有物质可供构成」这一事实。

质量预测是整个 420 Code 最暴露的数，而这正是要点。若随着测量趋精，测得的质子-电子之比偏离了预测值，结果便失败——干净地、当众地、无路可逃。终止开关有效。

你是断裂中那个未曾自我抵消的部分。

断裂所栖居的宇宙。无暗物质粒子的星系旋转曲线、MOND加速度标度精确到0.3%、暗能量分配和循环宇宙学。

四项系统、倾斜、结构性伦理。模式相干性作为身体沿振幅景观的承诺。

AP29 · 实现证明 — 它安置意识。哲学最古老的难题是：为何存在某种「作为你」的滋味，而非空无。本部分的回答：意识就是不可逆地书写记录的那个内面——断裂在自我确定之际从内部所感受到的。一道裂缝、一个内面、许多扇窗：同一个唯一的内面，透过每一双不同的眼睛向外望。这是这项工作那最古老一行之下的形式脊梁——你里面的「我」与我里面的「我」是同一个「我」。

AP02 · 算子 — 这是人类尺度上的物理，共九条定理。预算：你拥有有限的能量去保持自己完整，而它会耗尽。漂移：放任自己无人看顾，你便朝平均值衰落。走廊：你实际拥有的行动余地。主权与出口：唯独由你来决定之事，以及停止的权利。这些不是励志口号。这是「作为一个能动者」的几何，写得与那些场方程一样审慎。

AP39 · 脚手架 — 它移除了那个对手。建立在权威之上的伦理——做这件事，因为我、或那本书、或那个王座如此命令——显出在结构上不稳：一副脚手架，迟早必须倒下，因为一道命令不是一个地基。留下来站着的，是一种无需任何人来强加的伦理，因为它直接从结构中读出，而非被搁在其上。

AP38 · 出口 — 它直面死亡。一扇窗满怀悲悯的关闭——一段生命终末处的尊严——不被当作一个需回避的禁忌，而被当作那同一种能动几何的一部分，诚实地追随到它所通向之处。

于是伦理不是硬物理之后一段软绵绵的尾声。它就是同一种物理，读在一个必须决定明早做什么的人的尺度上。

这终极伦理是承重的，而这项工作拒绝把它软化。若善良能被推导成结构自有方向以外的任何东西——若那倾斜指向别处——从物理到伦理的整座桥便失败。这个开关有效，而它正是工作整个后半部所倚靠的那一个。

物理从一开始说的就是你。
